# 沖繩旅遊租借服務網站規格書

## 1. 網站概述
本網站旨在提供沖繩地區的旅遊租借服務，包括衝浪板、車輛、帳篷等項目，以方便快速的服務為特色，提升使用者的旅遊體驗。

## 2. 頁面結構分析 (index.html)

### 2.1 頁首 (Header)
- **Logo 區域**:
  - 包含一個連結至首頁的 Logo 圖片 (`okinawa.svg`)。
  - 網站標題文字：「沖繩旅遊租借服務」。
- **導覽列 (Navigation)**:
  - **桌面版**: 顯示橫向選單。
  - **行動版**: 透過漢堡選單 (`.toggle`) 切換顯示/隱藏。
  - **選單項目**:
    1. 首頁 (Home)
    2. 關於我們 (About Us)
    3. 會員登入 (Login)
    4. 聯絡我們 (Contact Us)

### 2.2 主要內容 (Main Content)
- **主視覺區塊 (Hero Section)**:
  - 背景圖片 (`T03.jpg`) 搭配濾鏡效果。
  - 宣傳標語：「準備好享受沖繩熱情了嗎? 現在開始準備」。
  - 行動呼籲按鈕 (CTA Button)：「現在開始」。

- **服務特色介紹 (Features Section)**:
  - 標題：「透過給予最好的服務以及品質，我們幫助你獲得最好的 沖繩旅遊經驗」。
  - **資訊卡片區塊 (Cards)**:
    1. **快速運送**:
       - 圖片 (`T11.JPG`)
       - 描述：承諾一個工作天內送達沖繩群島境內。
       - 連結：「運送地點」
    2. **付款方便**:
       - 圖片 (`T02.jpg`)
       - 描述：支援信用卡、現金卡、現金、銀行轉帳等。
       - 連結：「了解更多」
    3. **種類多樣**:
       - 圖片 (`T07.jpg`)
       - 描述：超過3000樣商品，通過品質篩檢。
       - 連結：「線上搜尋」

### 2.3 頁尾 (Footer)
- **Logo 區域**:
  - 包含黑色版本的 Logo (`black okinawa.svg`)。
  - 網站標題。
- **頁尾導覽列**:
  - 英文選單連結：Home, About, Login, Contact。
- **社群媒體連結 (Social Media)**:
  - Facebook
  - Instagram
  - Twitter
  - YouTube

## 3. 技術規格
- **HTML 版本**: HTML5
- **語言設定**: zh (繁體中文內容)
- **字型**: Google Fonts (Noto Sans TC)
- **樣式表**: `./assets/css/style.css`
- **腳本**: `./assets/js/script.js` (處理行動版選單切換)
- **RWD 響應式設計**: 針對移動裝置優化 (Breakpoint: 800px)
